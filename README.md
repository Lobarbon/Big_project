# Team7 2020 SOA Project

## Usage
Please run `bundle install` at the very first time.

## Language of the Domain
original JSON description -> our YAML description
- title -> title
- sourceWebPromote -> `website`
- showInfo -> `infos`
- time -> `start_time`
- endTime -> `end_time`
- location -> `address`
- locationName -> `location`
