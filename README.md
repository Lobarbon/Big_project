# IndieLand
Get the latest information of Taiwan Indie Music Events.

## Usage
Please run `bundle install` at the very first time.

## Language of the Domain
original JSON description -> our YAML description
- title -> `event name`
- sourceWebPromote -> `website`
- showInfo -> `sessions`
    - time -> `start_time`
    - endTime -> `end_time`
    - location -> `address`
    - locationName -> `place`
